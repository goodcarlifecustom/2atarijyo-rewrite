# AGENTS.md

## 目的

このリポジトリは、WordPress記事をCodex Cloudで自動リライトするためのものです。

ユーザーが記事URLを渡した場合、以下を確認待ちなしで最後まで実行してください。

- 元記事HTMLの取得
- SEOリライト
- 公的・信頼性のある外部リンクの追加
- SWELL向け装飾
- HTML検証
- WordPressへの新規下書き作成
- 作業ログの記録

## 最重要ルール

- Codex Cloud前提で作業すること。Codex CLI前提の説明や手順にしない。
- ユーザーが記事URLを渡したら、確認待ちで停止しない。
- `rewrite-plan.md` の作成だけで止まらない。
- 既存の重要なH2/H3を勝手に削除しない。
- 元記事より情報量を減らさない。
- 元記事より本文文字数を少なくしない。
- 検索意図を変えない。
- WordPressに貼り付け可能なHTMLで作成する。
- SWELLテーマで崩れにくいHTMLにする。
- WordPressの既存公開記事は直接更新しない。
- WordPressには必ず「新規下書き」として作成する。
- `.env` を作成・編集・コミットしない。
- 認証情報やアプリケーションパスワードを出力しない。
