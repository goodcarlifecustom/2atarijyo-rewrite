# SEOリライト基本ルール

## 絶対ルール

- 元記事の検索意図を変えない
- 既存の重要なH2/H3を勝手に削除しない
- 本文リライト前に必ずリライト計画を作る
- Codex Cloudで記事URLが渡された場合は、リライト計画作成後に確認待ちで止まらず本文リライトまで続行する
- 記事全体の情報量を減らさない
- 競合上位記事の構成を丸写ししない
- 読者にとって不要な説明は増やさない
- アフィリエイトリンクへの導線は自然にする
- 公共性や信頼性のある外部リンクを必要に応じて追加する
- 内部リンクを必要に応じて追加する
- h2/h3/h4の階層を崩さない
- WordPressに貼り付け可能なHTMLで出力する

## リライトの進め方

1. 元記事を分析する
2. リライト計画を作る
3. Codex Cloudで記事URLが渡された場合は確認待ちせず本文をリライトする
4. 変更内容を記録する
5. WordPress反映前のチェックを行う
